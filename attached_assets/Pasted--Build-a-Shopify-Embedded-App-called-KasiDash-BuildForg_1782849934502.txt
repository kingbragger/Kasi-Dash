# Build a Shopify Embedded App called KasiDash & BuildForge

Build a production-ready Shopify embedded application called **KasiDash & BuildForge**.

## Tech Stack

Use:

* Next.js 15 (App Router)
* TypeScript
* Shopify App Bridge
* Shopify Polaris
* Shopify Admin API
* Prisma ORM
* PostgreSQL (Neon or Supabase)
* Tailwind CSS
* Vercel compatible deployment
* Node.js runtime
* Environment variables
* ESLint
* Responsive UI

The project must run perfectly on **Vercel** without modifications.

---

## Authentication

Implement Shopify OAuth.

Store:

* Shop domain
* Access token
* Installation date
* Merchant information

Automatically reconnect merchants after installation.
